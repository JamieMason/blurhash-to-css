# blurhash-to-css

> Convert a blurhash string to a CSS object

## ðŸŒ© Installation

```
npm install blurhash-to-css
```

## ðŸ•¹ Usage

```ts
import { blurhashToCss } from "blurhash-to-css";

const css = await blurhashToCss({
  blurhash: "eCF6B#-:0JInxr?@s;nmIoWUIko1%NocRk.8xbIUaxR*^+s;RiWAWU",
  height: 400,
  width: 600,
});
```
